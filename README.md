# Campus Bookstore Checkout System (Java)

This project simulates a simple campus bookstore checkout system built in Java.

The system models different store items such as textbooks and merchandise and allows them to be added to a shopping cart for checkout.

Concepts used in this project:
- Object Oriented Programming
- Interfaces
- Abstract Classes
- Enums
- Arrays
- Composition

Classes Included:
- Item (base class)
- Textbook
- Merch
- Cart
- CheckoutApp
- Category (enum)
- DiscountType (enum)
- Sellable (interface)

This project was created as practice for Java OOP concepts including inheritance, polymorphism, and interface design.
